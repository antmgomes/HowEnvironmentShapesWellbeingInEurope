# Greener, Denser, Hotter: How Environment Shapes Wellbeing in Europe

## Introduction:
This project examines the relationship between environmental conditions (Normalised Difference Vegetation Index, Population Density, and Average Mean Temperature) and human outcomes (Happiness Index, Good Health %, Life Expectancy and Suicide Rate) across European countries from 2015 to 2023.
